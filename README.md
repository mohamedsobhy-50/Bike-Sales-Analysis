# Bike-Sales-Analysis

This project analyzes bike sales based on:
- Income Groups (Low, Medium, High)
- Age Groups (Adolescent, Middle Age, Old)
- Commuting Distance
- Gender

Visualizations include:
- Bubble Chart: Income vs Distance vs Sales
- Combo Chart: Age Group vs Income Group
- Treemap: Sales distribution by Type of Bike and Income Group

Insights:
- Identifies key customer segments
- Shows how distance affects bike purchases
